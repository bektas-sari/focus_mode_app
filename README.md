# 🧘 Focus Mode App

A modern and minimal Flutter app designed to help users stay focused by starting a countdown timer. Built with a clean purple theme and circular progress animation, it’s ideal for deep work sessions and improving productivity.

---

## 🚀 Features

- ⏱️ Start a focus timer with a single tap  
- 🎚️ Adjustable time (15–90 minutes) via slider  
- 🔄 Circular progress with countdown  
- 🟣 Sleek Material 3 interface  
- 📵 Distraction-free focus experience
